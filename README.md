# Dalail Khairat Assets

CDN page assets for the Dalailul Khairat reader in the Durood app.

Expected structure:

```text
pages/page-001.png
pages/page-002.png
manifests/dalail.json
```

CDN page URL pattern:

```text
https://cdn.jsdelivr.net/gh/SahilHasnain/dalail-khairat-assets@main/pages/page-{page}.png
```

The source PDF is intentionally not committed to this repository.
